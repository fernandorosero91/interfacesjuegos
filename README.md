# 🎮 Presentación Interactiva: Diseño de Interfaces para Videojuegos

Una presentación educativa en formato de videojuego estilo arcade retro para la exposición universitaria de Ingeniería de Software.

## 🚀 Inicio Rápido

1. Abre el archivo `index.html` en tu navegador
2. Haz clic en "INICIAR JUEGO"
3. ¡Listo para presentar!

## 🎯 Cómo Usar Durante la Exposición

### Opción 1: Navegación Automática (Recomendado)
1. **Haz clic en cualquier zona de color** en el mapa
2. Pacman se desplazará automáticamente hacia esa zona
3. Cuando llegue, se abrirá el panel educativo
4. El jugador correspondiente expone su tema
5. Clic en "CONTINUAR" para cerrar el panel
6. Repite para las siguientes zonas

### Opción 2: Botón "Siguiente Zona"
- Haz clic en el botón "➜ SIGUIENTE ZONA" en el HUD
- Pacman irá automáticamente a la siguiente zona en orden

### Opción 3: Control Manual
- Usa las flechas del teclado (⬆️⬇️⬅️➡️) para mover a Pacman
- Presiona ESPACIO para activar el turbo

## 🎮 Controles

- **Flechas del teclado:** Mover a Pacman manualmente
- **Espacio:** Activar turbo
- **Clic en zonas:** Navegación automática
- **Botón "Siguiente Zona":** Ir a la siguiente zona en orden
- **Botón "Continuar":** Cerrar panel educativo

## 🏆 Finalización

El juego termina automáticamente cuando se visitan las 5 zonas educativas. Se muestra una pantalla de victoria con:
- Puntuación final
- Niveles completados
- Pellets recolectados
- Combo máximo alcanzado

## 💡 Consejos para la Presentación

1. **Proyecta en pantalla grande** para que todos vean claramente
2. **Usa la navegación automática** (clic en zonas) para mayor fluidez
3. **Cada jugador hace clic en su zona** cuando le toca exponer
4. **Los pellets son opcionales** - el objetivo principal son las zonas educativas
5. **Los fantasmas no atacan** durante la navegación automática
