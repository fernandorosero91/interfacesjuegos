Desarrolla una PRESENTACIÓN INTERACTIVA EN FORMATO VIDEOJUEGO para explicar el tema:

DISEÑO DE INTERFACES PARA VIDEOJUEGOS

La presentación será utilizada en una exposición universitaria de Ingeniería de Software, por lo que debe ser visualmente atractiva, didáctica, moderna y altamente interactiva.

El resultado final debe parecer un videojuego educativo completamente funcional.

-----------------------------------------------------

TECNOLOGÍAS

Construir el proyecto utilizando:

HTML
CSS
JavaScript

Opcionalmente se puede usar:

Canvas API
TailwindCSS
Animaciones CSS
Sprites retro
Audio arcade

El proyecto debe ejecutarse simplemente abriendo el archivo index.html.

-----------------------------------------------------

ESTILO VISUAL

Inspiración en videojuegos retro arcade.

El diseño debe incluir:

Fondo negro estilo arcade
Laberinto estilo videojuego
Colores neón (azul, amarillo, rojo)
Tipografía pixel retro
Animaciones suaves
Interfaz clara para proyectar en clase

Debe parecer un videojuego educativo retro.

-----------------------------------------------------

ELEMENTOS DEL JUEGO (OBLIGATORIOS)

El videojuego debe incluir obligatoriamente:

🎮 Un personaje principal que se mueve por el mapa usando las flechas del teclado.

⭐ Puntos o pellets distribuidos en el mapa que desaparecen cuando el personaje los recolecta.

👻 Fantasmas o enemigos que se mueven dentro del laberinto.

📊 Un sistema de SCORE que aumente cuando el jugador recolecta puntos.

📚 Ventanas emergentes educativas que aparezcan cuando el jugador entra en ciertas zonas del mapa y que expliquen el contenido de la exposición.

-----------------------------------------------------

INTERFAZ (HUD)

Debe existir un panel tipo HUD que muestre:

Score
Nivel actual
Indicador de progreso
Instrucciones de control

ESTRUCTURA DE NIVELES

Cada nivel del juego representa una parte de la exposición.

NIVEL 1

Tema:
¿Cómo ayudan las interfaces a mejorar la experiencia de los usuarios?

Contenido educativo:

Las interfaces permiten que los jugadores comprendan el funcionamiento del juego fácilmente.
Facilitan la navegación dentro del sistema.
Permiten tomar decisiones rápidas.
Mejoran la interacción entre el usuario y el juego.
Aumentan la inmersión del jugador.

Presentador:
Valentina

-----------------------------------------------------

NIVEL 2

Tema:
Ventajas para los usuarios

Contenido:

Facilidad de uso.
Acceso rápido a información importante.
Mejor comprensión de las mecánicas del juego.
Experiencia más agradable para el jugador.

Presentador:
Fernando

-----------------------------------------------------

NIVEL 3

Tema:
Buenas prácticas en el diseño de interfaces

Contenido:

Simplicidad visual.
Consistencia en los elementos de la interfaz.

Uso correcto de colores.
Accesibilidad para diferentes tipos de usuarios.

Presentador:
Vanesa

NIVEL 4

Alejandro

NIVEL FINAL

Pantalla de conclusión.

Mensaje:

Las interfaces son fundamentales en los videojuegos porque permiten que los jugadores interactúen con el sistema, comprendan la información y disfruten una experiencia clara y divertida.
